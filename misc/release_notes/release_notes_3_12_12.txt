RELEASE NOTES FOR KOHA 3.12.12
24 Mar 2014
========================================================================

Koha is the first free and open source software library automation package 
(ILS). Development is sponsored by libraries of varying types and sizes, 
volunteers, and support companies from around the world.  
The website for the Koha project is 

    http://koha-community.org/

Koha 3.12.12 can be downloaded from:

    http://download.koha-community.org/koha-3.12.12.tar.gz

Installation instructions can be found at:

    http://wiki.koha-community.org/wiki/Installation_Documentation
    
    OR in the INSTALL files that come in the tarball


Koha 3.12.12 is a bugfix/maintenance release.


It includes 3 bugfixes.

Critical bugs fixed in 3.12.12
======================


Notices
----------
	9210	major	Automatic carriage return are missing in print letters generated for holds


Other bugs fixed in 3.12.12
======================


I18N/L10N
----------
	11505	normal	Untranslatable table headers in Label batch manager

Templates
----------
	11502	minor	Branch filter in "Holds To Pull" is too fuzzy

Documentation
======================

documentation is 

    http://koha-community.org/documentation/

Revision control notes
======================

The Koha project uses Git for version control.  The current development 
version of Koha can be retrieved by checking out the master branch of 

    git://git.koha-community.org/koha.git

The branch for this version of Koha and future bugfixes in this release line is 3.12.x.

The last Koha release was 3.14.0, which was released on November 21, 2013.
